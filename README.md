# React Wordle clone

This project is the clone version of wordle game, which is implemented with React and Typescript.


## CLI commands

To run development

```bash
    npm run dev
```

To build project
```bash
    npm run build
```

To preview project
```bash
    npm run preview
```


## Environment/Tools
- React
- Vite
